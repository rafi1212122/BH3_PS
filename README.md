### Running the thing
1. Do Git Clone / Download as Zip

```bash
git clone https://github.com/rafi1212122/BH3_PS.git 
```

2. Make sure you have MongoDB server running

3. Install required packages

```bash
npm install 
```

4. Start the server
```bash
npm run dev
```
