### Running the thing
- Do Git Clone / Download as Zip

``` git clone https://github.com/rafi1212122/BH3_PS.git ```

- Copy Environment Variables

``` cp .env.example .env ```

- Migrate Database

``` npx prisma migrate dev ```

- Run npm install

``` npm install ```

- Run npm run dev

``` npm run dev ```
